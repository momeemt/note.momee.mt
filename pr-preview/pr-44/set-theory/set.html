集合

範囲のはっきりしたものの集まりを集合という。 集合は普通、ラテン大文字で表される。 幾何学的表現を用いて、集合を空間、その元を点と呼ぶこともある。

元

A が1つの集合であるとき、A に含まれる個々のものを A の元（元素、要素とも）という。 もの a が集合 A の元であることを、a∈A、あるいは A∋a で表す。 このことを、a が A に属する、a は A に含まれる、A は a を含むなどという。

a∈A の否定は、a∈A、または A∋a で表す。

a∈A と a∈A はいずれか一方だけが成り立ち、両方同時に成り立つことや、両方同時に成り立たないことはない。

無限に多くの元を持つ集合を無限集合、有限個の元しか持たない集合を有限集合という。 また固有名詞的に、自然数全体の集合を N、整数全体の集合を Z、有利数全体の集合を Q、実数全体の集合を R で表す。

A が集合であるとき、どんなもの a を取ってきても a∈A であるか a∈A であるかは定まっているが、その判定問題が容易に解けるかどうかはまた別問題。

変数と条件

考察の対象となるものを代表的に表す文字を変数という。 変数が表す対象は任意で、数値だけではない。

また、変数を含む文章をその変数についての条件または性質という。 条件 C が変数 x についての条件であることを明示したいとき、C(x) と表す。 ある具体的なもの a を C(x) に代入して得られる文章 C(a) が正しいとき、a は条件 C を満たす、または a は性質 C をもつ、という。 条件 C を満たすようなもの全体は1つの集合を形成する。

記法

一般に、元 a,b,c,⋯ より成る集合を以下のように表す。 これを外延的記法という。

{a,b,c,⋯}

ただし、この記法はすべての元を列挙するか、簡単に ⋯ 部分が推測できるような集合でなければならない。 そこで、ある条件を満たすもの全体の集合、ある性質を満たすもの全体の集合を表現できる、内包的記法を用いる。 条件 C を満たすようなもの全体の集合を、次のように表す。

{x∣C(x)}

たとえば、a,b を両端とする R の閉区間は {x∣a≤x≤b}、開区間は {x∣a<x<b} である。

空集合

元を全く含まない集合を空集合という。 空集合は ∅、あるいは外延的記法では {} と表す。 たとえば、{x∣x∈R,x2=−1} は空集合。

空集合 ∅ は元を含まないから、∀a​a∈∅ である。

相等

集合 A,B は、全く同じ元から成るとき、すなわち以下が成り立つときに等しいという。 これを、A=B と表す。

∀x​(x∈A⇔x∈B)

同等

2つの文章 p,q が与えられたとき、p→q とは、p が正しいときに q もまた正しいことを示す。 また、q→p も成り立つとき、p⇔q と表せて、p と q は（論理的に）同等であるという。

部分集合

集合 A,B において、以下を満たすならば A は B の部分集合であるといい、A⊂B または B⊃A と表す。 これを、A は B に含まれる、B は A を含むという。

∀x​(x∈A→x∈B)

その否定は、A⊂B、または B⊃A と表される。

A⊂B∧A=B であるとき、A は B の真部分集合であるという。

A=B であるための必要十分条件は、A⊂B∧A⊃B が成り立つことである。 もし2つの集合が同等であることを証明したければ、両向きに部分集合であることを示す。

A⊂B∧B⊂C→A⊂C が成り立つことも明らか。

空集合と部分集合

空集合 ∅ は任意の集合 A の部分集合である。

∅⊂A

これは、含意を用いて証明できる。 ∅⊂A を示すためには、∀x​x∈∅→x∈A を示す必要がある。 ここで、∀x​x∈∅ であるから、常に正しい。 したがって、元の式は正しい。

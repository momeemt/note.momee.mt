共通部分

Definition 4.2.2.1 (共通部分).

2つの集合 A,B があるとき、A,B の両方に共通な元全体の集合

A∩B={x∣x∈A,x∈B}

を A と B の共通部分という。

記号 ∩ は交わり、meet、capなどと呼ばれる。 たとえば、A を正の偶数全体の集合、B を正の奇数全体の集合とすると、A∩B=∅。

一般に、A∩B=∅ である時には、A,B は交わるという。 また、A∩B=∅ である時には、A,B は交わらない、または互いに素であるという。

基本的な性質

包含関係

Proposition 4.2.2.1 (共通部分の包含関係).

A⊃A∩B,B⊃A∩B

Proposition 4.2.2.1 (共通部分の包含関係) の証明

任意に x∈A∩B を取る。 Definition 4.2.2.1 (共通部分)より、x∈A かつ x∈B。

よって、x∈A より、A⊃A∩B また、x∈B より、B⊃A∩B。□

上位集合閉包

Proposition 4.2.2.2 (共通部分の上位集合閉包).

A⊂C,B⊂C→A∩B⊂C

Proposition 4.2.2.2 (共通部分の上位集合閉包) の証明

任意に x∈A∩B を取る。 Definition 4.2.2.1 (共通部分)より、x∈A かつ x∈B。 また、前提より A⊂C から、x∈C。 したがって、A⊂C,B⊂C→A∩B⊂C が示された。

ここから、A∩B は、A,B の両方に含まれる集合のうちで最大のものであることが言える。

冪等律

Proposition 4.2.2.3 (共通部分の冪等律).

A∩A=A

Proposition 4.2.2.3 (共通部分の冪等律) の証明

任意に x∈A∩A を取る。 Definition 4.2.2.1 (共通部分)より、x∈A かつ x∈A、すなわちx∈A。 よって ∀x​(x∈A∧x∈A→x∈A) を満たすから、A∩A⊂A が示された。

次に、任意に x∈A を取る。 A∩A⊃A を示すために、∀x​(x∈A→x∈A∧x∈A) を示したい。 ここで、x∈A∧x∈A は x∈A であるから、自明に示される。

したがって、Proposition 4.2.2.3 (共通部分の冪等律)は示された。

交換律

Proposition 4.2.2.4 (共通部分の交換律).

A∩B=B∩A

Proposition 4.2.2.4 (共通部分の交換律) の証明

任意に x∈A∩B をとる。 Definition 4.2.2.1 (共通部分)より、x∈A かつ x∈B。 ∀x​(x∈A∧x∈B→x∈B∧x∈A) を満たすなら A∩B⊂B∩A だが、これは自明。

同様に、任意に x∈B∩A をとる。 Definition 4.2.2.1 (共通部分)より、x∈B かつ x∈A。 ∀x​(x∈B∧x∈A→x∈A∧x∈B) を満たすなら B∩A⊂A∩B だが、これも自明。

したがって、A∩B=B∩A は示された。

結合律

Proposition 4.2.2.5 (共通部分の結合律).

(A∩B)∩C=A∩(B∩C)

Proposition 4.2.2.5 (共通部分の結合律) の証明

任意に x∈(A∩B)∩C をとる。 Definition 4.2.2.1 (共通部分)より、以下のように変形できる。

x∈(A∩B)∩C​→x∈A∩B∧x∈C→x∈A∧x∈B∧x∈C→x∈A∧x∈B∩C​

よって、∀x​(x∈A∩B∧x∈C→x∈A∧x∈B∩C) を満たすから、 (A∩B)∩C⊂A∩(B∩C) が正しい。

同様に、任意に x∈A∩(B∩C) を取ると、x∈A∩B∧x∈C に変形できる。 よって、∀x​(x∈A∧x∈B∩C→x∈A∩B∧x∈C) を満たすから、 A∩(B∩C)⊂(A∩B)∩C が正しい。

したがって、(A∩B)∩C=A∩(B∩C) が示された。

多クラス分類を行う際に利用する回帰。

Softmax関数

Sigmoid関数を多ラベルに拡張した関数をSoftmax関数という。 Softmax関数の j 番目の出力は以下のようになる。

σ(z)j​=∑d​exp(zd​)exp(zj​)​

全ての成分が0以上で、和が1になるK次元ベクトル全体の集合をK-1シンプレクスという。 K-1シンプレクス上の任意の点は、K次元確率ベクトルである。

k∑​σ(z)k​=1

Softmax関数は RD をK-1シンプレクスに写像する。

Softmax回帰

1-of-k表記（1 hot vector）の訓練データを用いて、以下の予測 t^nk​ を計算する。

t^nk​=Pr(k∣xn​)=∑k=1K​exp(wkT​xn​)exp(wkT​xn​)​

次に、訓練データ集合全体に対する尤度 L(w) を計算する。

L(w)=n=1∏N​k=1∏K​Pr(k∣xn​)tn,k​=n=1∏N​k=1∏K​(t^n,k​)tn,k​

最後に、誤差関数（交差エントロピー）を得る。

E(w)=−logL(w)=−n=1∑N​k=1∑K​tnk​logt^nk​
